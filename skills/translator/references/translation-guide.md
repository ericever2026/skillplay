# 翻译参考指南

## 中英翻译规则

### 词性转换
| 中文 | 英文 |
|------|------|
| 名词化 | Nominalization |
| 动词化 | Verbalization |
| 形容词化 | Adjectivization |

### 常见句式对照

#### 时间表达
| 中文 | 英文 |
|------|------|
| 今天 | today |
| 昨天 | yesterday |
| 明天 | tomorrow |
| 上周 | last week |
| 下个月 | next month |
| 每年 | every year / annually |

#### 数量表达
| 中文 | 英文 |
|------|------|
| 很多 | many / a lot of |
| 一些 | some / a few |
| 很少 | few / little |
| 没有 | no / none |

| 中文 | 英文 |
|------|------|

## 翻译质量标准

1. **准确性**：正确传达原文含义
2. **流畅性**：符合目标语言表达习惯
3. **一致性**：术语翻译保持统一
4. **完整性**：不遗漏关键信息
